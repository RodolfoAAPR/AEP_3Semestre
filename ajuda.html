Como Usar o Portal

Guia completo para navegar e aproveitar todos os recursos do Portal Maringá

Página Inicial

Navegação Principal

Use o menu superior para acessar as diferentes seções:

  • Início: Página principal com visão geral dos serviços
  • Transporte: Informações sobre ônibus, rotas e horários
  • Saúde: Localização de postos de saúde e unidades médicas
  • Educação: Escolas e creches públicas
  • Contato: Informações para entrar em contato

Busca Rápida

Na seção "Acesso Rápido", você pode:

  • Digite o nome de uma rua ou bairro para encontrar serviços próximos
  • Use o botão "Usar Localização" para encontrar serviços perto de você
  • Clique em "Horários" para ver os horários de funcionamento

Transporte Público

Planejando sua Viagem

  1. Acesse a página "Transporte" no menu principal
  2. Use o "Planejador de Viagem" inserindo origem e destino
  3. Clique em "Buscar Rota" para ver as opções disponíveis

Encontrando Pontos de Ônibus

  • Na seção "Pontos Próximos", clique em "Usar Minha Localização"
  • Permita o acesso à localização quando solicitado
  • Veja a lista de pontos mais próximos com distâncias

Informações das Linhas

Cada linha de ônibus mostra:

  • Número da linha: Identificação única
  • Rota: Origem e destino principal
  • Frequência: Intervalo entre ônibus
  • Status: Se a linha está ativa

Unidades de Saúde

Encontrando Postos de Saúde

  1. Clique em "Saúde" no menu principal
  2. Use o mapa interativo ou a lista de unidades
  3. Filtre por tipo de atendimento ou especialidade
  4. Veja endereços completos e horários de funcionamento

Informações Disponíveis

  • Endereço completo e como chegar
  • Telefone para contato
  • Horários de funcionamento
  • Especialidades médicas disponíveis
  • Serviços oferecidos

Educação

Escolas Públicas

Encontre informações sobre:

  • Localização das escolas municipais
  • Níveis de ensino oferecidos
  • Contatos para matrícula
  • Documentação necessária

Creches e Educação Infantil

  • Vagas disponíveis por região
  • Processo de inscrição
  • Documentos necessários
  • Critérios de seleção

Usando no Celular

Navegação Mobile

  • Toque no ícone de menu (☰) no canto superior direito
  • O menu se abrirá com todas as opções disponíveis
  • Toque fora do menu para fechá-lo
  • Todos os botões são otimizados para toque

Recursos Mobile

  • Localização: Permita o acesso para encontrar serviços próximos
  • Ligações: Toque nos números de emergência para ligar diretamente
  • Zoom: Use gestos de pinça para ampliar textos e mapas

Acessibilidade

Recursos de Acessibilidade

  • Navegação por teclado: Use Tab para navegar entre elementos
  • Textos grandes: Interface otimizada para leitura fácil
  • Cores contrastantes: Melhor visibilidade para todos
  • Ícones intuitivos: Símbolos universais para facilitar compreensão

Dicas para Melhor Experiência

  • Use o zoom do navegador se precisar de textos maiores
  • Ative o leitor de tela se necessário
  • Os botões têm descrições claras para facilitar navegação
